# Labs

## Get all labs

This endpoint retrieves all labs.

### HTTP Request

`GET /api/labs`

### Query Parameters

Parameter | Default
--------- | -------
api_key   | null

## Get one lab

This endpoint retrieves a specific lab.

### HTTP Request

`GET /api/labs/:id`

### Query Parameters

Parameter | Default
--------- | -------
api_key   | null
lab_id    | null
