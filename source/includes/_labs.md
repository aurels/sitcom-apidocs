# Labs

## Get all labs

This endpoint retrieves all labs.

### HTTP Request

`GET /api/labs`

### Query Parameters

Parameter | Type   | Required
--------- | ------ | --------
api_key   | String | Yes

## Get one lab

This endpoint retrieves a specific lab.

### HTTP Request

`GET /api/labs/:id`

### Query Parameters

Parameter | Type    | Required
--------- | ----    | --------
api_key   | String  | Yes
id        | Integer | Yes
